My experience lies in Microsoft .NET desktop applications, including C#, ASP.NET Web API, WPF, and SQL Server.

Driven by a passion for modern technology, I am currently transitioning into full-stack web development using Blazor.

I leverage cutting-edge AI tools such as GitHub Copilot and ChatGpt app (Codex) to optimize development workflows, supported by a formal certification in Generative AI for Software Developers.
